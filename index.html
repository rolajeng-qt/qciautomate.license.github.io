📋 使用說明

本系統提供 API 服務用於生成軟體許可證。請按照以下步驟使用本地端應用程式：

  1. 1
    下載本地端應用程式
    下載 LicenseActivator.exe 到您的電腦
  2. 2
    執行應用程式
    雙擊執行 LicenseActivator.exe，程式會自動：
    • 獲取您電腦的硬體ID
    • 向服務器請求生成許可證
    • 下載並保存許可證文件
  3. 3
    獲取許可證文件
    成功執行後，將在程式目錄下生成 license.lic 許可證文件
